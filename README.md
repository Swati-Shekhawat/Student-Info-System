# Student-Info-System

Project Title : Student Informantion System

Description   : This simple web-based application build to manage student records digitally. 
                The main goal of this project is to provide an  easy way to register new 
                students and show their information such as name, roll number, email in a
                secure database. Instead of using manual paperwork, this system automates
                the process using a clean we interface.
                
Technologies used:
1. Fronted: HTML and CSS for a user-friendly registration form.
2. Backend: PHP for server-side processing and logic.
3. Database: MYSQL to store and manage student data.
